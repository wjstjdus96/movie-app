## ⚙️ 기능 설명

- Home & Tvshows 
  - theMovieDatabase API를 사용하여 데이터 패칭
  - 카테고리 별로 구분된 영화&시리즈 데이터 출력
  - 슬라이더 활용
  - 슬라이더 아이템 클릭 시 모달을 통한 상세설명 출력
- Search
  - 헤더 우측 돋보기 아이콘을 통해 키워드 검색
  - Search 페이지에서 키워드 검색
  - 카테고리(전체, 영화, 시리즈) 필터를 통한 선택적 검색 가능
